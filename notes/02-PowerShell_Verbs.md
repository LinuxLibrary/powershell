# PowerShell Verbs

- To create something new

	`New`

- To list or show

	`Get`

- Change some characteristics

	`Set`

- Remove something

	`Remove`

- Get all the verbs

	`Get-Verb`

---

# Below are some examples

- To rename a VM

	`Rename-VM -Name "VM1" -NewName "WebVM1"`

- To rename computer

	`Rename-Computer -Name "PC24" -NewName "AccountsPC"`
